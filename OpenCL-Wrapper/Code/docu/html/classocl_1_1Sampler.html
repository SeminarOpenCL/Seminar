Constructor & Destructor Documentation

ocl::Sampler::Sampler ( )
explicit

Instantiates this Sampler with the active Context.

If no Context is provided the active Context of the Platform is taken if there is an active Context.

The OpenCL Sampler is not created. Do not forget to create the OpenCL Sampler.
