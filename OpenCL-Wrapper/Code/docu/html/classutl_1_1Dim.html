Detailed Description

Utility class to store the shape of multidimensional structures.

The shape of a multidimensional structure is stored within a vector which store the size of each dimension. A shape can be known or unkown. In case of a known shape all vector entries must be natural numbers. If there is an unkown dimension the vector entry of the corresponding element is equal -1. There are predefined matrix shapes such as scalars, vectors or matrices. Any shape vector with more 2 dimension entries is called multidimensional array.

bool utl::Dim::isMatrix ( ) const

Prüft ob die Dimensionsgrößen die eines Vektors entsprechen.

Definition eines Vektors:
Alle Dimensionsgrößen müssen einer 1 entsprechen, ausser die beiden ersten Richtungen, die größer 1 oder variabel sein müssen.

  • Beispiel für die Form einer Matrix: Dim=N,6 oder Dim=N,M.
  • Keine Matrix: Dim=1,N,M oder Dim=3,1,N.

Achtung : Die variablen Größen könnten einer 1 entsprechen, wodurch keine Matrixform gegeben wäre, sondern einem Vektor oder Skalar entspreche.

Definition eines Vektors:
Alle Dimensionsgrößen müssen einer 1 entsprechen, ausser die erste oder zweite Richtung, die größer 1 oder variabel sein kann.
