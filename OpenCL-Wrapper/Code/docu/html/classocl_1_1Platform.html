Detailed Description

Wrapper for cl_platform_id.

The Platform model consists of a host connected to one or more Device objects. A Device is divided into one or more compute units (CUs) which are further divided into one or more processing elements (PEs). Computations on a Device occur within the processing elements. Note that within one Platform different DeviceTypes might exist.
A Platform has mutiple Context objects aggregating the Device objects within the Platform. There can only be one active Context which can be switched any time in the program flow. Within one Context multiple Queue objects can be created. One of these must be selected as an active Queue on which commands are enqueued. Multiple Platforms may exist.
