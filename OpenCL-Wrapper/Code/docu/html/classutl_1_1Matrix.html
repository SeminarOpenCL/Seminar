Detailed Description

template<class T, class F>
class utl::Matrix< T, F >

MatrixBase class which corresponds to a two-dimensional array.

An Args object can be instantiated in order to conveniently store the arguments from the console. Arguments are strored as strings which can be converted to other data types.
