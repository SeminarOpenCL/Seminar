Detailed Description

Utility class to profile functions.

The ProfilePass class can be inherited to profile a function. Objects from inherited classes of ProfilePass can be stored in a ProfilePassManager object which runs all ProfilePass functions. Inhertied objects need to implement the prof and ops functions.
