Detailed Description

Wrapper for cl_mem. Base class of the Buffer and Image class.

In order to create Memory object an active Contex must exist. The Memory object inserts itself automatically into the Context, while the liveness depends on the range of the scope of its definition. When the scope is left, the Memory object destructs itself and removes itself from its Context. When a Context leaves its scope all corresponding Memory objects will also be destructed and the Memory becomes invalid. The buffer and image class are using the Memory class for common functions. Do not instantiate this class.
